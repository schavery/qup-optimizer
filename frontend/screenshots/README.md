# Screenshots Directory

Add your webview screenshot here as 'webview.png'

To capture a screenshot:
1. Open http://localhost:5001 in your browser
2. Set up your desired layout with nodes placed
3. Take a screenshot (Cmd+Shift+4 on macOS, Win+Shift+S on Windows)
4. Save as webview.png in this directory

Recommended dimensions: 1920x1080 or similar 16:9 aspect ratio
